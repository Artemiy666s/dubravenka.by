# DUBRAVENKA — Летнее кафе

Монолитный сайт-меню для летнего кафе DUBRAVENKA.

## Запуск

```bash
npm install
npm start
```

Сайт откроется на http://localhost:3000

## Тесты

```bash
npm test
```

## Админ-панель

Открывается напрямую по адресу (кнопки на сайте нет):

**http://localhost:3000/admin**

Пароль по умолчанию: `dubravenka`  
Задать свой: переменная окружения `ADMIN_PASSWORD`

В панели можно редактировать:
- главную (hero, промо-сет)
- меню, сеты, напитки, акции
- страницу «О нас» и галерею
- контакты и карту
- SEO-настройки

Данные сохраняются в `data/site.json`.

## Структура

- `index.html` — главная страница (SPA)
- `admin/` — панель управления
- `data/site.json` — все данные сайта
- `css/styles.css` — стили
- `js/app.js` — навигация и интерактив
- `js/menu-data.js` — резервные данные (fallback)
- `assets/svg/` — SVG-иллюстрации, иконки, логотип

## Особенности

- Быстрая загрузка: только SVG, без тяжёлых изображений
- Адаптивный дизайн (mobile-first)
- Нижняя навигация: Меню, Акции, О нас, Контакты
- Все разделы меню по макету
